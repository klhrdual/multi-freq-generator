# Platinum ToneGen

A premium, platinum-styled multi-frequency audio generator with advanced harmonic synthesis, real-time visualization, and internationalization support.

## Features

- **Multi-Frequency Generation**: Create complex soundscapes by layering multiple oscillators.
- **Harmonic Synthesis**: Built-in instrument presets (Piano, Violin, Cello, Oboe, etc.) that automatically generate up to 10 harmonics based on real-world acoustic data.
- **Visualizer**: Real-time oscilloscope visualization of the audio output.
- **Themes**: Switch between "Platinum" (Light) and "Obsidian" (Dark) modes.
- **Localization**: Full support for English and Traditional Chinese.
- **Sharing**: Share your exact frequency configuration via URL.
- **Sorting**: Organize frequencies ascending, descending, or by creation order.
- **Responsive Design**: optimized for both desktop and mobile devices.

## Tech Stack

- **React 18**: UI Framework.
- **TypeScript**: Type safety.
- **Tailwind CSS**: Styling and Dark Mode.
- **Web Audio API**: High-performance audio synthesis.
- **Lucide React**: Iconography.

## Usage

1. **Add a Tone**: Click the `+` button.
2. **Select Base**: Choose a base note/frequency and waveform.
3. **Select Instrument (Optional)**: Choose an instrument preset to automatically generate harmonics.
4. **Play**: Click the Play button to start the audio engine.
5. **Adjust**: Modify frequencies, waveforms, or volume levels in real-time.
6. **Share**: Click the Share icon to copy a link to your current configuration.
